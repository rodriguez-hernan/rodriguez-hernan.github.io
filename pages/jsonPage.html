In this section I am using the Star Wars API to get information about the characters. There is a list provided of 87 characters, so with this app we can look at the information available. Just fill in a number and look at the JSON object we receive.
I also added 2 buttons to check the Species and World Home of the selected character.
To be able to do this, I had to parse the data into an object, retreive the corresponding data and make a new request.
