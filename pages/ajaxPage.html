AJAX

In this page you can interact with these 3 buttons. The first one calls an external api called SWAPI swapi.co that retrieves data of characters of Star Wars.
The second button, makes also an AJAX request, but the function fetches a local json from this project.
In both cases, the data is processed and displayed in the screen.
The last button is there to clean up the screen.
NOTE: if you open the console of the browser (F12), you can see the data we are obtaining.
